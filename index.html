Dayz-installer

DayZ-Installer is an installer (and updater) for DayZ (mod of ARMA II Combined Operations Modification).

View the Project on GitHub Narno/DayZ-Installer

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Features

  • Online installer (and updater, obviously)
  • Detects ARMA II and ARMA II Operation Arrowhead (based on Windows Registry, 32 and 64 bits) and its installation directory
  • Choice of server download mirror
  • Detects the DayZ Mod and compare version with the last available on the default CDN
  • Checks MD5 sums of each RAR archive
  • Creates a Steam app desktop shortcut

Requirements

To play the DayZ Mod

  • ARMA II: Combined Operations (Get it on Steam)

To use DayZ-Installer

  • An Internet connection (DayZ-Installer is an online installer)
  • (7zip EXE (and DLL): included in DayZ-Installer, with its licence)

To compile DayZ-Installer yourself

  • Last version of NSIS (Nullsoft Scriptable Install System) compiler
  • NSIS plugins:
    • Inetc plug-in
    • MD5 plugin
